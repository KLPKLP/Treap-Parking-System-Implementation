# Treap-Parking-Management-Implementation

Semestrální projekt z předmětu Datové struktury a algoritmy vytvořený v jazyce C#.

Projekt je zaměřen na evidenci obsazených parkovacích míst v parkovacím domě s využitím uživatelsky implementované abstraktní datové struktury typu Treap. Každé podlaží parkovacího domu je reprezentováno samostatnou instancí této struktury.

Aplikace umožňuje obsazení a uvolnění parkovacího místa, ověření obsazenosti konkrétního stání, výpis obsazených míst ve vzestupném pořadí a nalezení nejbližšího volného místa k požadovanému číslu. Součástí řešení je také základní grafické uživatelské rozhraní pro testování jednotlivých funkcí aplikace.

## Funkcionalita
- evidence obsazených parkovacích míst podle podlaží,
- obsazení konkrétního parkovacího místa,
- uvolnění obsazeného místa,
- ověření, zda je místo obsazené,
- výpis obsazených míst ve správném pořadí,
- nalezení nejbližšího volného místa,
- demonstrace zachování vlastností datové struktury Treap při vkládání a mazání prvků.

## Použité technologie
- C#
- .NET
- Windows Forms

## Cíl projektu
Cílem projektu je ukázat praktické využití datové struktury Treap při návrhu aplikace pro správu parkovacích míst v parkovacím domě.
